Soleil v1.1 - Beta Release
============================

Three-Voice Harmony Generator
Built with Faust

HOW TO INSTALL
--------------

1. Unzip this file
2. Drag Soleil_v1.1.app to your Applications folder (or anywhere)
3. Right-click (or Control-click) on Soleil_v1.1.app
4. Select "Open" from the menu
5. Click "Open" when macOS asks if you're sure
6. That's it! After the first time, just double-click to open

WHAT IT DOES
------------

Soleil takes a mono audio input and creates a three-voice harmony in stereo:
- Dry signal (your original audio)
- Harmony 1 (default: octave up)
- Harmony 2 (default: perfect 5th up)

Perfect for guitar, vocals, synths, and more!

DEFAULT SETTINGS
----------------
- Dry Gain: 34%
- Harmony 1: +12 semitones (octave up), 33% mix
- Harmony 2: +7 semitones (perfect 5th), 33% mix

CONTROLS
--------
- Window Size: Adjusts quality vs latency (default: 2048 samples)
- Crossfade: Smoothness of transitions (default: 256 samples)
- Each voice has interval selector and gain control

SYSTEM REQUIREMENTS
-------------------
- macOS 10.13 or later
- Audio interface or built-in audio
